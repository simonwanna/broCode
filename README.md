# BroCode

## Project Setup
This project uses [uv](https://github.com/astral-sh/uv) for dependency management.

Install dependencies:
```bash
uv sync
```

## Usage

### UI
Run the Streamlit app:
```bash
uv run streamlit run ui/app.py
```

### Repo Graph
Index a repository:
```bash
uv run repo-graph --help
```
